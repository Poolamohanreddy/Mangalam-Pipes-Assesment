# Mangalam HDPE Pipes Product Page

This project follows the provided Figma assignment as a vanilla HTML, CSS, and JavaScript implementation.

## How to Open

Open `index.html` in a browser.

## What Is Included

- Product-detail page matching the Figma direction for Mangalam HDPE Pipes.
- Header and navigation with a sticky header that appears after the first fold.
- Product image carousel with arrows and thumbnail controls.
- Hover/focus zoom preview for the main carousel image.
- Catalogue request card with email input.
- Applications section with next/previous controls.
- Responsive desktop, tablet, and mobile layout.

## Files

- `index.html` - page structure
- `styles.css` - visual design and responsive layout
- `script.js` - sticky header, carousel, zoom, form, and application controls
- `assets/` - local SVG image assets
